# 🥋 Sistema de Apoyo a la Planificación - Club UNSU

Sistema de Inteligencia Artificial para optimizar la logística de torneos de Karate-Do mediante predicción de duración de categorías.

## 🎯 Características

- **Precisión**: R² = 97.77% | MAE = 31.85 segundos
- **Modelo**: Random Forest entrenado con 744 registros históricos
- **Interfaz**: Streamlit low-code para usuarios no técnicos
- **Explicabilidad**: LIME para auditar predicciones

## 🚀 Instalación
```bash
# Clonar repositorio
git clone https://github.com/tuusuario/CAPSTONEESTIMACIONKARATE.git
cd CAPSTONEESTIMACIONKARATE

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar aplicación
streamlit run app.py
```

## 📊 Resultados

- **Reducción de retrasos**: 85% (de 2-4 horas a 25-30 minutos)
- **Error relativo**: 7.19% (objetivo: <15%)
- **Validación cruzada**: MAE = 34.65 ± 5.50 segundos

## 🏆 Impacto

- Optimiza planificación de 680+ participantes anuales
- Reduce sobrecostos de alquiler de instalaciones
- Mejora experiencia de atletas, jueces y familias
